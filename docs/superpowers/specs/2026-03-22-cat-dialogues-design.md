# Cat Dialogue System — Design Spec

**Date:** 2026-03-22
**Feature:** 고양이 종별 상황별 대사 시스템

---

## Overview

각 고양이 종마다 집사와의 관계 단계에 따라 3가지 대사를 표시한다.
발견 횟수 기반으로 상황이 자동 결정되며, "목표 고양이 발견!" 화면과 고양이 도감 두 곳에서 표시된다.

---

## 상황 매핑

| 발견 횟수 | 상황 | 라벨 |
|-----------|------|-------|
| 1회 | 첫만남 | `첫만남` |
| 2회 | 친해지기 | `친해지기` |
| 3회 이상 | 집사되기 | `집사되기` |

---

## UI 설계

### 말풍선 컴포넌트
- 고양이 이미지 위/옆에 말풍선 오버레이
- 상단: 상황 라벨 (작은 뱃지) + 고양이 이름
- 본문: 대사 텍스트
- 스타일: 둥근 말풍선, 꼬리가 고양이 방향을 가리킴

### result 화면 ("목표 고양이 발견!")
- 고양이 이미지 렌더링 후 300ms delay로 말풍선 CSS fade-in (`opacity: 0` → `opacity: 1`, transition 0.4s)
- 발견 횟수는 `getCatCount(catId)` (score.js)로 조회 — play.js에서 `addToCollection` 호출 후 result로 이동하므로 이미 카운트에 반영되어 있음
- 발견 횟수 기준으로 상황 자동 결정 (count=1 → 첫만남, count=2 → 친해지기, count≥3 → 집사되기)

### 고양이 도감 (showCatDetail 팝업 내부)
- 기존 `showCatDetail` 팝업 오버레이 **내부**에 말풍선 추가 (별도 팝업 X)
- 팝업 하단 영역에 말풍선 + 상황 라벨 표시
- 미발견 고양이(`getCatCount === 0`)는 말풍선 미표시
- 기존 뱃지 라벨 `'친구'`(count=2) → `'친해지기'`로 통일

## 데이터 구조

```js
// src/game/catDialogues.js
export const CAT_DIALOGUES = {
  catId: {
    first:    '첫만남 대사',
    friendly: '친해지기 대사',
    butler:   '집사되기 대사',
  }
}

export function getDialogue(catId, discoveryCount) {
  const d = CAT_DIALOGUES[catId];
  if (!d || discoveryCount === 0) return null; // 미발견 고양이는 null
  const situation = discoveryCount === 1 ? 'first' : discoveryCount === 2 ? 'friendly' : 'butler';
  const label = discoveryCount === 1 ? '첫만남' : discoveryCount === 2 ? '친해지기' : '집사되기';
  return { situation, label, text: d[situation] };
}
```

## 41종 전체 대사

### Common (기본 합성 타일)

**코리안숏헤어 (korean)** — 활발하고 솔직한 길고양이 에너지
- 첫만남: "...뭐야, 나 왜 쳐다봐. 간식 있어?"
- 친해지기: "어, 또 왔네. 뭐 가져왔어?"
- 집사되기: "나 배고파. 밥 줘. 지금 당장."

**러시안블루 (russian)** — 낯을 가리지만 한 번 정들면 평생 충성
- 첫만남: "...가까이 오지 마세요. 아직 당신을 모르니까요."
- 친해지기: "...당신은 조용하군요. 나쁘지 않아요."
- 집사되기: "당신 옆이 제일 평화로워요. 거기 계속 있어요."

**봄베이 (bombay)** — 검은 표범 외모, 강아지 같은 애교
- 첫만남: "나 불러? 나야 나! 완전 잘생겼지~"
- 친해지기: "나 따라다닐게. 어디 가? 나도 같이 가!"
- 집사되기: "집사야~ 집사야~ 집사야~! (무한반복)"

**벵갈 (bengal)** — 야생적이고 운동신경 폭발, 에너지 무한
- 첫만남: "...지금 뛰어갈 준비 중이야. 방해하지 마."
- 친해지기: "같이 놀자! 근데 규칙은 내가 정해."
- 집사되기: "나 오늘 벽 열 번 타고 물건 두 개 떨어뜨렸어. 기록이야."

**렉돌 (ragdoll)** — 안으면 힘 빠지는 초온순, 집사 그림자
- 첫만남: "...안아줘도 돼. 도망 안 가."
- 친해지기: "여기 앉아도 돼? 무릎 빌려줄게."
- 집사되기: "어디 가? 나도... (따라감)"

### Rare (S1–S11)

**스코티시폴드 (scottish)** — 접힌 귀, 올빼미 눈, 침착한 관찰자
- 첫만남: "...흠. (동그란 눈으로 빤히 봄)"
- 친해지기: "당신, 나쁘지 않네요."
- 집사되기: "오늘도 여기 앉겠습니다. 문제 있나요?"

**페르시안 (persian)** — 풍성한 털, 왕족 자존심, 느긋한 여유
- 첫만남: "...무릎 준비됐나요? 내가 앉아드릴 수도 있어요."
- 친해지기: "빗질 잘하던데요. 계속해도 돼요."
- 집사되기: "이 집은 이제 내 왕국이에요. 당신은 충신이고요."

**먼치킨 (munchkin)** — 짧은 다리, 폭발적 에너지, 끝없는 호기심
- 첫만남: "안녕! 나 작아 보여도 엄청 빠르거든?"
- 친해지기: "저기 높은 곳은 어떻게 올라가? 방법 알려줘!"
- 집사되기: "다리 짧은 게 장점이야. 무게중심이 낮잖아."

**브리티시숏헤어 (british)** — 품위 있고 독립적, 안기기 싫어함
- 첫만남: "접근 허가는 아직 안 났어요."
- 친해지기: "...같은 방에 있어도 되긴 해요."
- 집사되기: "안아들지 않는다면, 당신 옆에 있어드리죠."

**아메리칸숏헤어 (american-sh)** — 사교적이고 적응력 좋은 가족형
- 첫만남: "어, 안녕! 나 적응 빠른 편이야. 금방 친해질 수 있어."
- 친해지기: "이 집 이제 익숙해졌어. 집사도 마음에 들고."
- 집사되기: "가족이 생겼다! 우리 가족!"

**샴 (siamese)** — 세상에서 제일 수다스러운 고양이
- 첫만남: "야옹. 야옹. 야옹. 야옹. 보고 있어? 야옹."
- 친해지기: "나 오늘 할 말 많아. 들을 준비 됐어? 야옹야옹야옹~"
- 집사되기: "집사가 조용히 있으면 안 돼. 내가 채워줄게. 야옹야옹!"

**노르웨이숲 (norwegian)** — 자연파 등산가, 풍성한 이중 털
- 첫만남: "...숲에서 왔어. 여기도 나쁘지 않네."
- 친해지기: "높은 데 같이 올라갈래? 나 잘 가르쳐줄 수 있어."
- 집사되기: "겨울에 내 털 만져봐. 천연 핫팩이야."

**버미즈 (burmese)** — 사교적이고 사람을 사랑하는 비단결 고양이
- 첫만남: "안녕하세요! 저 사람 만나는 거 엄청 좋아해요!"
- 친해지기: "오늘 같이 낮잠 잘 수 있어요? 저 옆에 있어도 돼요."
- 집사되기: "집사님이 최고예요. 제 인생에서 제일 잘한 선택이에요."

**엑조틱숏헤어 (exotic-sh)** — 페르시안처럼 느긋, 납작한 얼굴, 영원한 졸음
- 첫만남: "...zzz... 어, 왔어요? zzz..."
- 친해지기: "여기... 편해요... 계속 있어도 돼요... zzz..."
- 집사되기: "깨워줘서 고마워요... 밥 줄 거죠? zzz..."

**터키시앙고라 (turkish-angora)** — 우아하고 활동적, 물을 좋아하는 귀족
- 첫만남: "처음 뵙겠습니다. 제 우아함에 놀라지 마세요."
- 친해지기: "저 물 좋아해요. 같이 수도꼭지 틀어줄 거죠?"
- 집사되기: "이 집에서 제일 아름다운 건 저예요. 동의하죠?"

**아비시니안 (abyssinian)** — 지칠 줄 모르는 탐험가, 호기심 폭발
- 첫만남: "이 집 구석구석 다 탐험했어! 비밀 없어!"
- 친해지기: "저기 저 선반 올라갈 수 있어 봐. 못 말려."
- 집사되기: "집사, 오늘 같이 탐험 몇 번 했어? 나는 열두 번."

### Epic (S12–S27)

**히말라얀 (himalayan)** — 페르시안 몸에 샴 포인트, 조용하고 온순
- 첫만남: "...조용히 있을게요. 방해하지 마세요."
- 친해지기: "당신 손 따뜻하네요. 조금만 더 만져도 돼요."
- 집사되기: "페르시안의 품위와 샴의 감성. 그게 저예요."

**하바나브라운 (havana-brown)** — 희귀 초콜릿색, 발로 세상 탐색
- 첫만남: "(발로 콕콕) 이게 뭐야? 저게 뭐야? 흠..."
- 친해지기: "(발로 얼굴 콕) 이게 집사야? 흠, 나쁘지 않네."
- 집사되기: "세상 모든 걸 발로 확인해봤는데, 집사가 제일 좋아."

**소말리 (somali)** — 여우 같은 얼굴, 풍성한 꼬리, 장난기 가득
- 첫만남: "여우 아니야. 고양이야. 혼동하지 마."
- 친해지기: "이 꼬리 예쁘지? 만지고 싶으면 달리기 해봐."
- 집사되기: "집사는 내 꼬리 만질 자격이 있어. 특별히 허락해줄게."

**발리니즈 (balinese)** — 샴의 장모 버전, 우아하고 수다스러움
- 첫만남: "안녕하세요~ 드릴 말씀이 정말 많아요. 앉으세요."
- 친해지기: "오늘 제 이야기 들어줘서 고마워요. 내일도 있어요."
- 집사되기: "말이 많은 건 당신이 좋아서예요. 계속 들어줄 거죠?"

**메인쿤 (mainecoon)** — 젠틀자이언트, 강아지 같은 충성, 풍성한 수염
- 첫만남: "크다고 무서워하지 마. 젠틀자이언트야."
- 친해지기: "문 따라다니는 게 취미야. 집사 가는 곳마다 갈 거야."
- 집사되기: "내가 크니까 집사 보호는 내가 해. 걱정하지 마."

**오리엔탈숏헤어 (oriental-sh)** — 큰 귀 안테나, 날카로운 눈매, 수다쟁이
- 첫만남: "이 귀 봐. 안테나야. 집사 생각 다 읽혀."
- 친해지기: "지금 뭐 생각해? 말 안 해도 알아. 간식 생각이지?"
- 집사되기: "집사 마음 다 알아. 그래도 말해줘. 듣는 게 좋아서."

**이집션마우 (egyptian-mau)** — 점박이 무늬, 집고양이 중 최속, 충성스러움
- 첫만남: "저 세상에서 제일 빠른 집고양이예요. 믿어요?"
- 친해지기: "저만 따라오면 돼요. 빠르게 갈게요."
- 집사되기: "집사는 제 파라오예요. 영원히 모실게요."

**재패니즈밥테일 (japanese-bt)** — 짧은 꼬리, 행운의 고양이, 활발
- 첫만남: "만복이야~ 행운 가져다줄게. 꼬리 짧은 게 포인트야."
- 친해지기: "나랑 있으면 좋은 일만 생겨. 써봐서 알아."
- 집사되기: "집사한테만 특별히 최고 등급 행운 드릴게. 감사해요~"

**데본렉스 (devon-rex)** — 요정 같은 얼굴, 큰 귀, 장난꾸러기
- 첫만남: "요정이야? 고양이야? 둘 다야! 나야 나!"
- 친해지기: "나 장난쳐도 화내지 마. 사랑해서 그러는 거야."
- 집사되기: "집사 인생에 마법이 필요하지? 내가 있잖아."

**샤르트뢰 (chartreux)** — 회청색, 말 없는 철학자, 미소 짓는 얼굴
- 첫만남: "...(동그란 눈으로 조용히 바라봄)"
- 친해지기: "말은 안 해도 마음은 전달되고 있어요."
- 집사되기: "우리 사이엔 말이 필요 없잖아요."

**코니시렉스 (cornish-rex)** — 곱슬곱슬 털, 에너지 무한, 따뜻한 곳 집착
- 첫만남: "만져봐! 곱슬곱슬하지? 신기하지?"
- 친해지기: "따뜻해서 좋아. 집사 무릎 자리 예약할게."
- 집사되기: "겨울엔 내가 집사 핫팩 해줄게. 공평하게."

**사바나 (savannah)** — 서벌 하이브리드, 야생미, 강아지급 충성심
- 첫만남: "야생에서 왔어. 여기가 마음에 들지는 두고 봐야 알아."
- 친해지기: "...인정. 여기 나쁘지 않아. 집사도 봐줄 만해."
- 집사되기: "내 충성심은 강아지급이야. 집사를 선택했어, 평생."

**맹크스 (manx)** — 꼬리 없음, 토끼처럼 깡충깡충, 독특한 걸음걸이
- 첫만남: "꼬리 없는 거 알아. 물어보려 했지? 태어날 때부터야."
- 친해지기: "내가 뛰는 거 봤어? 토끼처럼 깡충깡충이야."
- 집사되기: "꼬리로 감정 표현 못 하는 대신 눈으로 해. 보여?"

**아메리칸컬 (american-curl)** — 뒤로 말린 귀, 온순하고 장난기 있음
- 첫만남: "귀가 뒤로 말렸지? 태어날 때부터 패션이야."
- 친해지기: "이 귀 만져도 돼. 살살만요."
- 집사되기: "집사 귀도 말려있으면 더 잘 어울릴 텐데. 아쉽다."

**토이거 (toyger)** — 호랑이 줄무늬 집고양이, 활발하고 친근함
- 첫만남: "나 호랑이야. 아니, 고양이야. ...호랑이처럼 생긴 고양이야."
- 친해지기: "줄무늬 세어봤어? 나도 모르겠어. 같이 세볼까?"
- 집사되기: "집사는 나의 정글이야. 여기가 제일 좋아."

**터키시반 (turkish-van)** — 수영하는 고양이, 물 사랑, 이색눈
- 첫만남: "물 좋아하는 고양이 봤어? 나야."
- 친해지기: "욕조 같이 써도 돼? 수영 가르쳐줄게."
- 집사되기: "집사가 목욕할 때 옆에 있어줄게. 외롭지 말라고."

### Legendary (S28–S36)

**옥시캣 (ocicat)** — 오셀롯 무늬, 완전 집고양이, 당당한 자신감
- 첫만남: "야생 같지? 사실 집에서 태어났어. 반전이지."

**싱가퓨라 (singapura)** — 세계 최소 고양이, 큰 눈, 넘치는 호기심

## 발견 횟수 추적

`score.js`의 `getCatCount(catId)` 함수로 발견 횟수 조회 (0~3, COLLECTION_MAX=3으로 캡).
- count=0: 미발견 → 말풍선 미표시
- count=1: 첫만남
- count=2: 친해지기
- count≥3: 집사되기

`addToCollection(catId)`은 play.js에서 이미 호출되므로 result.js/collection.js에서 재호출 금지.
